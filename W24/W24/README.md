# Module 9 Week 24 - Advanced Topics: Typescript and intro to Formik

## install typescript
```
npm install
```

## Run the typescript compiler
```
npm run tsc
```
`Note: we added this command manually in package.json`

## install React app dependencies
```
cd my-account && npm i
```

## start React app
```
cd my-account && npm start
```
